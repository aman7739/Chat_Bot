
# College Enquiry Chatbot

A Flask-based chatbot that answers college-related inquiries using machine learning.

## Features
- NLP-powered intent recognition
- Instant college admission queries resolution
- PDF knowledge base integration
- Interactive web interface

## Tech Stack
- **Backend:** Flask (Python)
- **ML:** scikit-learn, TF-IDF vectorization
- **Frontend:** HTML, CSS, JavaScript

## Setup
1. Clone the repo
2. Install requirements: `pip install -r requirements.txt`
3. Run: `python app.py`
4. Visit `http://localhost:5000`

## Files
- `app.py` - Main Flask application
- `intents.csv` - Training data
- `model.pkl` - Trained ML model
- `vectorizer.pkl` - TF-IDF vectorizer
